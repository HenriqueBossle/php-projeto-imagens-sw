////Explicações\\\\

| Função                            | O que faz                                                                         |
| ----------------------------------| --------------------------------------------------------------------------------- |
| glob("imagens/*.jpg")             | Cria um array com os caminhos de todos os arquivos `.jpg` no diretório `imagens/` |
| file_get_contents("arquivo.json") | Lê o conteúdo de um arquivo                                                       |
| json_decode($conteudo, true)      | Converte o JSON em array associativo PHP                                          |
| array_rand($array)                | Retorna uma chave aleatória do array                                              |
| basename($caminho)                | Extrai apenas o nome do arquivo (sem o caminho)                                   |
